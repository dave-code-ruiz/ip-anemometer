Embed wind speed charts in WordPress content. For use with the free IP anemometer application for the Raspberry Pi, available on http://zieren.de

== Description ==
The IP anemometer applicaton available on http://zieren.de measures wind speed, weather and more using a Raspberry Pi. It also has some extra features useful at an RC model aircraft flying field.

This plugin provides shortcodes to easily embed wind speed (and other) charts in WordPress content. The charts will show the data gathered by your Raspberry Pi.
